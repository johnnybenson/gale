Zero Config Migration

Reads your existing .stylelintrc.json directly. /* stylelint-disable */ comments work as-is.

SCSS & Less

Full preprocessor support via the raffia parser. Lint .scss, .less, and .css files.

--fix

Autofix

Automatically fix 15 rules. Hex shortening, zero units, vendor prefixes, quotes, and more.

LSP

Editor Integration

Built-in LSP server for real-time diagnostics. Works with VSCode, Neovim, and any LSP client.

$

Caching & Parallel

File-based caching skips unchanged files. Parallel linting via rayon for maximum throughput.

Quick Start

# Install
$ npm install -D gale-lint
 
# Initialize config
$ gale --init
 
# Lint your project
$ gale .
 
# Autofix problems
$ gale --fix .
